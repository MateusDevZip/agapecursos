# Guia de Estilo - Ágape Cursos

## Paleta de Cores (Extraída da Logo)

### Cores Principais
- **Primary (Rosa Mauve)**: `#A76B7D` - Pétalas externas da flor de lótus
- **Primary Light (Rosa Médio)**: `#C88B9A` - Pétalas médias
- **Primary Lighter (Rosa Pêssego)**: `#E8B4B8` - Pétalas internas
- **Accent (Rosa Suave)**: `#F5D5D8` - Tom claro para destaques

### Cores de Fundo
- **Background Light**: `#FFF9F9` - Branco cremoso
- **Background Dark**: `#2D1B1E` - Marrom escuro para dark mode
- **Surface Light**: `#FFF3F4` - Rosa muito claro
- **Surface Dark**: `#3D2529` - Marrom médio para dark mode

### Cores de Texto
- **Text Main**: `#5C3A40` - Marrom escuro
- **Text Secondary**: `#8C5E66` - Rosa acinzentado
- **Text Muted**: `#A76B7D` - Rosa mauve (mesma cor primary)
